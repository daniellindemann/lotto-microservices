# Dapr

<https://dapr.io>

# Installation

-  Install DAPR CLI
    ```
    wget -q https://raw.githubusercontent.com/dapr/cli/master/install/install.sh -O - | /bin/bash
    ```

- Initialize DAPR containers
    ```
    dapr init
    ```





# Uninstall

- Uninstall completely
    ```
    dapr uninstall --all    # everything dapr related
    ```
